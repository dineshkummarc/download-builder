# download-builder 

## UNDER DEVELOPMENT (v0.0.0)

Download builder for jQuery UI

## Usage

``` js
  $ jquib [build-dir]
```

## Example

``` bash
  $ cd /some/build
  
  #
  # Make a package.json file
  #
  $ nano package.json
  
  #
  # Build that file
  #
  $ jquib
```

## Tests
All tests are written with [vows][0] and should be run with [npm][1].

``` bash
  $ npm test
```
